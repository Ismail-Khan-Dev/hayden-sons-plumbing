## 🏆 Key Architectural Pillars

### 1. High-Performance Motion System
Utilizing a hybrid of **GSAP** for complex scrolling orchestrations and **Framer Motion** for micro-interactions, the site feels "alive." Transitions are physically modeled to ensure zero jank and maximum tactile feedback.

### 2. Conversion-Focused Funnel
Every pixel is positioned with intent.
- **Hero-to-Call Transition**: Optimized for sub-second decision making.
- **Dynamic Service Catalog**: Interactive cards that provide information depth without overwhelming the user.
- **Emergency CTA**: A persistent, non-intrusive portal for high-urgency lead generation.

### 3. Mobile-First Responsiveness
We recognize that 85% of emergency plumbing searches happen on mobile. Our layout adapts fluidly, ensuring that buttons are thumb-friendly and typography remains legible under natural light.

| Layer | Technology | Purpose |
| :--- | :--- | :--- |

## 🚀 Deployment & Operations

### Local Development
```bash
# Clone the repository
git clone https://github.com/Ismail-Khan-Dev/hayden-sons-plumbing.git

# Install dependencies (Resetting lockfile for perfect environment sync)
npm install

# Launch Dev Portal
npm run dev
```

### Production Deployment
The infrastructure is optimized for **Vercel**. 
- **Continuous Deployment (CD)**: Every push to `main` triggers an atomic build.
- **Edge Caching**: Assets are delivered from the nearest geography to the user for <50ms TTFB.
